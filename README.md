# Football Team Manager (C Project)

This is a console-based football management system written in C.

## Teams Covered
- Real Madrid (2025/26 season)
- FC Barcelona (2025/26 season)

## Features
1. View team profile (name, manager, formation, trophies)
2. View full squad list with position and stats
3. View individual player stats by selecting a number

5. View injury report for each team
